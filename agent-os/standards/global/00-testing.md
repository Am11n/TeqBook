# Testing

- Vitest:
  - unit: rene funksjoner (utils, validation)
  - integration: services/repositories med mock/stub der det trengs
- Playwright:
  - kritiske user-flows
  - auth/booking/betaling der relevant

Krav
- Nye features skal ha minst én test på riktig nivå.
- Bugfix: reproducer med test når praktisk.
